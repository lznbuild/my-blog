<!-- 元组 -->
let tuple:[number,string] = [1,'str'];

let obj:object = {x:1,y:2}
不可修改obj的属性,不推荐这样使用



枚举enum的值只是可读的


private 不能被子类调用
protected 不能在实例中访问，子类可以
abstract 修饰属性和方法，必须在子类中实现的方法


### 索引类型

interface 只能约束class 的public成员

interface Obj{
  a:number,
  b:number
}

let value:Obj['a']   相当于number类型
